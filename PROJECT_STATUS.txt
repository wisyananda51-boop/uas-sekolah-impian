- Backend: Go 1.x, GORM 1.31.1, Gorilla Mux 1.8.1, Gorilla Handlers 1.5.2
- Database: MySQL 8.0 (via XAMPP)
- Build Tools: PostCSS, Autoprefixer

📊 DATABASE SCHEMA:
Created database: sekolah_impian

Table: facilities
- id (INT, PRIMARY KEY, AUTO_INCREMENT)
- title (VARCHAR)
- description (TEXT)
- icon_name (VARCHAR)

Table: programs
- id (INT, PRIMARY KEY, AUTO_INCREMENT)
- name (VARCHAR)

🚀 HOW TO RUN:

Option 1 - Automatic (Windows):
  → Double-click: start.bat
  → Servers akan auto-start di 2 windows terpisah

Option 2 - Manual:

Terminal 1 (Backend):
  cd backend
  go run main.go
  → Backend running di http://localhost:8080

Terminal 2 (Frontend):
  cd frontend
  npm run dev
  → Frontend running di http://localhost:5173

Access di browser: http://localhost:5173
